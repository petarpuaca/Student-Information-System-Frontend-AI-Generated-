# 🇷🇸 Serbian Language Chatbot

A web-based chatbot application focused on Serbian language interaction and conversational assistance.

This project was built as a frontend application with AI-assisted development, using modern React tooling and UI libraries.

---

## ✨ Features

- Chat-style user interface
- Serbian language-oriented interaction
- Clean and responsive frontend layout
- Component-based UI structure
- Fast local development and deployment workflow

## 🛠 Tech Stack

- **Frontend:** React
- **Language:** TypeScript
- **Build Tool:** Vite
- **UI:** shadcn/ui
- **Styling:** Tailwind CSS
- **AI-assisted frontend development:** Lovable

---

## 📂 Project Structure

- **src/** – application source code
- **public/** – static assets
- configuration files for Vite, Tailwind, TypeScript, and ESLint
